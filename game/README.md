# 🌊 海洋钓鱼 — H5 海洋生存游戏

最近更新：2026-05-02

一款融合**岸钓**、**大鱼吃小鱼**、**探险**、**PK 对战**四种玩法的 HTML5 海洋游戏，纯 Canvas 2D + 少量 DOM UI，无需任何外部依赖和构建工具，打开浏览器即可游玩。

---

## 玩法

### 岸钓模式
坐在岸边手持钓竿垂钓。拖拽屏幕调整抛竿力度，点击「抛竿」将鱼钩甩入大海。鱼钩沉入水中后水下的鱼会被吸引靠近，一旦咬钩自动收线拉回入账。

### 下海模式（大鱼吃小鱼）
点击「下海」化身一条小鱼潜入海洋。拖拽控制游动方向，吞噬比自己 tier 小或相等的鱼来成长升级。从浮游生物大小的 Lv.1 一路进化到能与虎鲸抗衡的 Lv.12。被更大的鱼碰到就会被吃掉，死亡后可选择重新开始，或花费 10 积分原地复活保持当前体型。

### 探险模式
时限奖励型探索关卡（`adventure.js`）。

### PK 模式
回合制吃鱼对战，双方轮流 30 秒一轮，共 3 轮决胜负。PK 得分记入单场 round score，不累计到总积分。

## 功能特色

### 82 种海洋生物
划分 **12 个 tier**，从浮游生物、磷虾等微生物，到小丑鱼、蝴蝶鱼、狮子鱼等观赏鱼，到金枪鱼、旗鱼、大白鲨等大型掠食者，直至虎鲸、鲸鲨、巨齿鲨、海怪利维坦等传说生物。每种鱼都有独特颜色、体型、速度、描述。

### 100 张真实素材图 + Atlas 切图系统
- 引入 1306×1204 的 10×10 atlas（`assets/fish-atlas.png`），用 Python + PIL 基于亮度分析自动切成 100 张独立 PNG 存放在 `assets/fish/`。
- `G.FISH_IMG_MAP` 映射 59 种 config 鱼到对应素材；未命中的鱼（浮游生物、水母、海怪利维坦等抽象生物）自动回退到 Canvas 写实绘制。

### 图鉴（Dex）系统
- 顶部按钮「图鉴」一键打开。
- 按 tier 分组展示全部 82 条鱼，已发现显示名字和素材图，未发现显示灰色剪影和 `???`。
- 未发现卡片点击弹出占位详情，引导玩家「下海探索并吃掉这种鱼来解锁它的资料」。
- 已发现卡片点击弹出完整详情（名字、tier、描述、积分/速度/尺寸）。
- **首次发现**任意鱼（海洋吃掉 / 岸钓钓起 / PK 吃掉）都会触发顶部金色横幅 `G.showDiscoveryFlash`，海洋和钓鱼模式额外奖励 2 倍积分，PK 模式不入总积分但会更新图鉴。
- 下海模式点击画面中的鱼直接复用图鉴详情弹窗，UI 一致。

### 写实化绘制（v2）
- 身体渐变接入 **countershading**（背深腹浅），模拟真实海洋鱼类的伪装特征。
- 新增 **鳃盖线 / 侧线 / 鳞片阵列 / 鳍条** 等写实细节原语。
- 条纹升级为带软化边缘的 linear gradient，去掉生硬黑边，与身体融合更自然。
- 眼睛三层结构：巩膜 + 瞳孔 + 高光。
- **鲨鱼族**（shark/tigerShark/hammerhead/greatwhite/whaleshark/megalodon）统一写实风格：渐变 body + 曲线鳃裂 + 背部暗带。
- **鲸类**（whale/orca/dolphin）写实化：黑白渐变 body + 喉部褶皱 + 呼吸孔 + 背部高光。
- **特殊形状鱼** 全部升级到渐变版：
  - **河豚** — 球形渐变 body + countershading + 细三角刺 + 去掉黑线
  - **狮子鱼** — 渐变身体 + 透明鳍膜扇 + 鳍尖发光点 + 软化条纹
  - **水母** — 半透明钟形 + 内部胃囊线 + 尾须渐隐触须
  - **章鱼** — 渐变触须 + 曲线吸盘 + 皮肤斑块 + 横瞳
  - **鮟鱇** — 球形渐变 + 疣突 + 发光饵 halo 外晕
  - **海怪克拉肯** — 渐变触须 + 密集吸盘 + 红色缝瞳 + 皮肤斑块

### 装备商店
积分可兑换 4 类装备：
- **鱼竿**（4 种）— 影响抛竿距离和收线速度，黄金竿拥有金色外观
- **鱼食**（4 种）— 提升鱼的吸引范围，秘制鱼饵让稀有鱼出现率翻倍
- **鱼钩**（4 种）— 扩大钩住范围，钛金钩额外加 20% 得分
- **角色**（5 种）— 渔夫、海盗、美人鱼、船长、自定义头像角色

### 背包系统
所有购买的装备收纳在背包中，随时查看和切换装备。

### 个人中心
自定义头像上传（localStorage base64）、昵称编辑，展示总积分 / 钓鱼数 / 下海最高 tier / 图鉴发现数等统计。头像会出现在钓鱼人脸上（岸钓时）或角色鱼的眼睛处（下海时）。

### 沉浸式海洋场景
多层渐变的天空与深海、动态焦散光纹、随水流摆动的海草群、珊瑚与岩石装饰、漂浮的浮游粒子、海底深处的雾气，以及三层叠加的海面波浪，全部由 Canvas 2D 实时绘制。

### 数据持久化
积分、装备、头像、昵称、图鉴发现记录、最高 tier、达到过的深度全部通过 localStorage 自动保存，关闭浏览器再打开进度不丢。

## 技术信息

- **技术栈**：纯 HTML5 + CSS + JavaScript，Canvas 2D 渲染 + 少量 DOM UI
- **外部依赖**：无（零依赖，零构建工具）
- **兼容性**：支持桌面浏览器和移动端触屏操作
- **代码规模**：约 3900 行 JS + 245 行 HTML

### 关键全局

- `G` — 所有模块挂在这上面
- `G.mode` — `'shore' | 'ocean' | 'pk' | 'adventure'`（图鉴缩略图绘制时临时设为 `'dex'` 避开 shore 的 0.55x 缩放）
- `G.OCEAN_FISH` — 82 种鱼数据源
- `G.FISH_SHAPES[shape]` — Canvas 绘制函数表
- `G.FISH_IMG_MAP` — 鱼名到素材图文件名的映射
- `G.discovered` — Set，已发现鱼名集合
- `G.FISH_SCALE` — 渲染缩放因子（shore 模式为 0.55）
- `G.discoverFish(fishType, opts)` — 发现入口，`{bonus:false}` 可关闭积分奖励（PK 用）
